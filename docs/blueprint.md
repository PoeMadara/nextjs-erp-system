# **App Name**: ERP Simplified

## Core Features:

- Login Page: Login page with username/password fields and a 'remember password' feature.
- Dashboard: Dashboard with key metrics display.
- CRUD Interface: CRUD interface for managing Customers, Products, Suppliers, and Employees.
- Warehouse Management: Display and manage warehouses.
- Invoice Management: Invoice management: viewing invoices

## Style Guidelines:

- Login page with a colorful background.
- General theme: white, gray, and black tones for an elegant design.
- Accent color: Blue (#3498db) for interactive elements and highlights.
- Sidebar navigation for main sections (Dashboard, Clients, Suppliers, etc.).
- Clean and organized table layouts for data display (Clients, Suppliers, Products, etc.).
